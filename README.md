## 📌 Project Overview
The **Library Management System** is a robust backend application designed to streamline the operations of a modern library. It manages book inventories, user registrations, and lending transactions using a relational database to ensure data integrity and efficient retrieval.

## 📢 Features

**✅ Inventory & Catalog Management**
* Full CRUD (Create, Read, Update, Delete) operations for managing books and digital resources.
* Advanced search functionality to filter books by author, genre, or ISBN.

**✅ User & Membership Handling**
* Secure user registration and authentication protocols.
* Automated tracking of borrowed items and member history.

**✅ Data Persistence & Integrity**
* Integrated **MySQL** database for reliable record-keeping.
* Implemented complex SQL queries for reporting and real-time availability checks.

**✅ Robust Software Design**
* Developed using strict **Object-Oriented Programming (OOP)** principles to ensure scalability.
* Comprehensive error handling to manage invalid inputs and database connection issues.

## 🛠 Tech Stack

**🖥 Language & Design**
* **Java:** Core language for application logic.
* **OOP Architecture:** Leveraging Inheritance, Encapsulation, and Polymorphism.

**⚙️ Database & Backend**
* **MySQL:** Relational database management.
* **JDBC:** For seamless Java-to-Database connectivity.

## 📐 System Architecture

This project utilizes a structured approach to separate data logic from business logic.



## 💻 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/rober-saliba/Library-Management-System.git](https://github.com/rober-saliba/Library-Management-System.git)
   ```
2.Database Setup:

Import the provided .sql schema into your local MySQL instance.

Update the DatabaseConfig.java file with your MySQL username and password.

3.Compile and Run:
   ```bash
   javac Main.java
   java Main
   ```
